## 使用方法

```shell
  conda create -n spyder python=3.9
```

### 安装依赖库

   ```shell
   pip install -r requirements.txt
   pip install playwright-stealth
   ```

### 安装 playwright浏览器驱动

   ```shell
   playwright install
   ```
